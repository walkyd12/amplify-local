# amplify-local

Local emulator for AWS Amplify Gen 2 — run your backend locally with zero AWS credentials.

Reads your `amplify/` TypeScript backend definitions and spins up local replacements for AppSync (GraphQL), DynamoDB, and S3. Generates an `amplify_outputs.json` your frontend consumes with zero code changes.
